# Конструктор интерфейса «Blog Customizer»
## Описание проекта
React-приложение для настройки внешнего вида блога. Пользователь может изменять цвета, размеры шрифтов, фон и другие параметры.

## Основной функционал
* Панель настроек.
* Применение стилей в режиме реального времени.
* Локальное состояние.
* Работа с формами.

## Используемые технологии
* React
* JavaScript
* CSS
* Работа с DOM через React

## Полученные навыки
* Управление состоянием компонента.
* Создание UI-инструментов.

## Запуск
npm install  
npm start
